区别：

  1. 函数表达式可以省略函数名，我们可以省略函数名创建匿名函数。
  2. 函数声明会被提升，与变量提升一样。函数声明会被提前到作用域的最前面，我们可以在声明的前面使用函数，而函数表达式则不可以：

说到函数表达式，必须提一下立即执行函数（Immediately-Invoked Function Expression, IIFE）。立即执行函数就是定义一个函数，然后立即执行。就像这样：function foo(){}()，括号用来立即执行函数。但是直接这么使用可能会产生语法错误。因为javascript解析器会把它这里的函数定义当做函数声明。如何让javascript解析器不把函数定义当做函数声明呢，那就是让解析器把它当做函数表达式。我们马上想到的方法就是加括号：
